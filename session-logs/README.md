# Session Logs

Store session logs here.

**Filename format:** `YYYYMMDD_HHMM_agent-name_short-summary.md`

Example: `20260201_0930_codex_inception-plan.md`

Each log should include:
- Duration
- Skills Used, when applicable
- Summary of work completed
- Decisions made
- Next steps

If this folder has no prior working session log, record "no prior active session log" in the next plan or session log and continue.
